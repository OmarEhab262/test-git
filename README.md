# React Experimentation Project

This project is a playground for experimenting with various React technologies, including:

- **React Context API**: Used for state management and theme switching.
- **Redux**: Exploring global state management.
- **React Query**: Handling asynchronous data fetching and caching.
- **TypeScript**: Adding type safety to React components.
- **React Router**: Implementing client-side navigation.
